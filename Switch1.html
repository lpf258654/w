    • console口登录密码遗失

    1、通过其他登录方式如：telnet或stelnet登录重置密码

    登陆时认证方式是密码：

  • [HUAWEI]user-interface console 0
  • [HUAWEI-ui-console0]authentication-mode password
  • [HUAWEI-ui-console0]set authentication password cipher xxhbgs2019
  • [HUAWEI-ui-console0]return
    • 登陆时认证方式是aaa:

  • [HUAWEI]user-interface console 0
  • [HUAWEI-ui-console0]authentication-mode aaa
  • [HUAWEI-ui-console0]q
  • [HUAWEI]aaa
  • [HUAWEI-aaa]local-user admin123 password irreversible-cipher xxhbgs2019
  • [HUAWEI-aaa]local-user admin123 service-type terminal
  • [HUAWEI-aaa]return

    • 2、通过bootrom菜单清楚密码（设备需重启）

  • Press Ctrl+B or Ctrl+E to enter BootLoad menu: 1
  • Password: -----Admin@huawei.com
  • Enter your choice(1-8): 7-----清除console口密码
  • Enter your choice(1-8): 1-----不能选择reboot，否则此次清除密码将失效
  • 执行清除Console密码的选项，其实是将Console口的认证方式设为None认证，设备重新启动后，不需要输入用户名和密码，可以直接登录设备。

    • 配置console登录认证方式及超时时间-----xxhbgs2019

  • [HUAWEI]user-interface console 0 -----进入console配置模式
  • [HUAWEI-ui-console0]authentication-mode password -----设置认证模式为password
  • [HUAWEI-ui-console0]set authentication password cipher xxhbgs2019 -----设置密码
  • [HUAWEI-ui-console0]idle-timeout 10 0 -----设置超时时间

    • 配置telnet登录认证方式

    配置密文登录

  • [HUAWEI]user-interface vty 0 4 -----进入模式，0 4表示允许5个并发链接
  • [HUAWEI-ui-vty0-4]authentication-mode password -----设置认证模式为password
  • [HUAWEI-ui-vty0-4]set authentication password cipher xxhbgs2019 -----设置密码
  • [HUAWEI-ui-vty0-4]user privilege level 3 -----设置用户等级3，缺省为1（0访问级1监控级2系统级3管理级）

    • telnet远程登录设备 192.168.0.1 255.255.255.0 admin123 xxhbgs2019

  • [HUAWEI]telnet server enable -----开启telnet服务
  • [HUAWEI]user-interface maximum-vty 8
  • [HUAWEI]user-interface vty 0 4
  • [HUAWEI-ui-vty0-4]protocol inbound telnet
  • [HUAWEI-ui-vty0-4]authentication-mode aaa
  • [HUAWEI-ui-vty0-4]q -----配置vty用户界面
  • [HUAWEI]aaa
  • [HUAWEI-aaa]local-user admin123 password irreversible-cipher xxhbgs2019
  • [HUAWEI-aaa]local-user admin123 service-type telnet
  • [HUAWEI-aaa]local-user admin123 privilege level 3
  • Warning: This operation may affect online users, are you sure to change the user privilege level ?[Y/N]y
  • [HUAWEI-aaa]q -----配置aaa认证

    • 删除vlan30

  • [HUAWEI]undo vlan 30 -----删除vlan30
  • [HUAWEI]dis vlan

    • 端口批量配置

  • [HUAWEI]port-group 1 -----创建端口组1，并进入端口视图
  • [HUAWEI-port-group-1]group-member GigabitEthernet 0/0/1 to GigabitEthernet 0/0/20 -----配置0/0/1到0/0/20加入端口组1
  • [HUAWEI-port-group-1]port link-type trunk -----配置端口类型为trunk
  • [HUAWEI-port-group-1]port trunk allow-pass vlan 30 -----配置端口允许vlan30通过
  • [HUAWEI-port-group-1]port link-type access -----配置端口类型为access
  • [HUAWEI-port-group-1]port default vlan 30 -----配置端口默认vlan30通过
  • Access类型的端口只能属于1个VLAN,一般用于连接计算机的端口。
  • Trunk类型的端口可以允许多个VLAN通过，可以接收和发送多个VLAN的报文，一般用于交换机之间连接的端口。

    • IPSG(ip和mac绑定)动态配置

    通过部署IPSG只能使用DHCP分配的IP方式，不允许私自配置地址，否则将无法访问网络

  • [HUAWEI]dhcp enable ------全局开启HDCP
  • [HUAWEI]dhcp snooping enable -----全局使用DHCP Snooping功能
  • [HUAWEI]vlan 30
  • [HUAWEI-vlan30]dhcp snooping enable -----vlan30开启HDCP
  • [HUAWEI-vlan30]dhcp snooping trusted interface gig 0/0/7 -----将连接DHCP的GE0/0/7设为信任接口
  • [HUAWEI-vlan30]ip source check user-bind enable -----基于vlan使用IPSG检查功能
  • [HUAWEI]dis dhcp static user-bind all 查看静态绑定表
  • [HUAWEI]display dhcp snooping user-bind all 查看DHCP Snooping动态绑定表

    • 配置DHCP Server和DHCP Relay

  • [HUAWEI]ip pool pool2 -----新增地址池pool2

  • [HUAWEI-ip-pool-pool2]dns-list 8.8.8.8 -----dns地址
  • [HUAWEI-ip-pool-pool2]lease day 1 hour 1 -----租用时间
  • [HUAWEI-ip-pool-pool2]q
